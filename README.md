## ⌨ Como executar o projeto

```bash
# Clonar o repositório
git clone https://github.com/rodrigodsluz/d1-test.git

# Entrar no diretório
cd d1-test

# Baixar as dependências
yarn install

# Executar o site
yarn start
```

Feito isso, abra o seu navegador e acesse `http://localhost:3000/`
